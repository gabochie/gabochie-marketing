# Deploy Live Sheet Population to GitHub Pages

## 🚀 Quick Deployment Steps

### **Step 1: Commit and Push**
```bash
git add live-sheet-populate.html
git commit -m "Add live sheet population tool"
git push origin main
```

### **Step 2: Enable GitHub Pages**
1. Go to your GitHub repository
2. Click **Settings** tab
3. Scroll to **Pages** section
4. Under **Build and deployment**, select **Deploy from a branch**
5. Select **main** branch
6. Click **Save**
7. Wait 2-5 minutes for deployment

### **Step 3: Access Live URL**
```
https://[your-username].github.io/Gabochie%20Marketing/live-sheet-populate.html
```

## 📋 What This Does

- **Bypasses local server issues** - No localhost needed
- **No CORS problems** - GitHub Pages handles cross-origin
- **Secure HTTPS** - Professional SSL certificate
- **Global access** - Works from anywhere
- **Direct Apps Script calls** - Most reliable method

## 🎯 After Deployment

1. **Test the live URL** - Should work immediately
2. **Click "🏗️ Initialize Sheet"** - Creates professional headers
3. **Click "🧪 Add Test Lead"** - Verifies system works
4. **Check your Google Sheet** - Should have headers with green background

## 📞 Troubleshooting

If you get 404:
- **Check file name** - Make sure it's exactly `live-sheet-populate.html`
- **Wait for deployment** - GitHub Pages takes 2-5 minutes
- **Check branch** - Must be deployed from `main` branch
- **Verify permissions** - Repository must be public
