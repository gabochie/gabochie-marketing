Cards

Cards use .card-bg background with var(--radius) and var(--shadow). Inner padding is 20px.

<div style="background:var(--card-bg);border-radius:var(--radius);box-shadow:var(--shadow);padding:20px;">
  <h2>Panel Title</h2>
  <p>Content...</p>
</div>

Filters use .filter-bar with flexbox layout. Search inputs and select dropdowns inside.

</div>

JavaScript Conventions

RuleStandard
Variable declarationvar for global/function-scoped, let/const for block-scoped
QuotesSingle quotes ('string') — avoid double quotes
SemicolonsAlways
Indentation2 spaces
Asyncasync function name() { ... } — not arrow async
Promisestry/catch preferred over .then()
Event handlersInline onclick attributes in HTML, named functions in JS
DOM accessdocument.getElementById() — not querySelector for IDs
Template literalsBackticks for string interpolation
ComparisonStrict === / !==

Apps Script (Code.gs) Conventions

RuleStandard
Indentation2 spaces
NamingcamelCase functions, UPPER_CASE constants
Private helpersTrailing underscore: getSheet_(), checkAuth_()
Sheet accessAlways via getSheet_() / getClientSheet_()
Return shape{ success: bool, data/error: ... }
Error handlingtry/catch wrapping every public function
doPost routingSwitch on data.action, each action maps to one function
AuthAll actions (except login/ping) call checkAuth_(token)
