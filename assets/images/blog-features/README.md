## Image Naming Convention

Use descriptive names:
- `blog-[category]-[description].jpg`
- Examples:
  - `blog-business-team-meeting.jpg`
  - `blog-tech-woman-laptop.jpg`
  - `blog-ghana-market-scene.jpg`

## Integration Instructions

1. Download images to `assets/images/blog-features/`
2. Optimize for web (max width 1200px, quality 85%)
3. Add alt text descriptions
4. Update blog posts to reference new images
